# asset-manager

## Development
### Requires:
- Python 3.7.1 (in virtualenv if possible)
- Mkdocs version 1.0.4

### Build:
- clone the repository
- `$cd  <repository>`
- `mkdocs serve` to test locally
- `mkdocs gh-deploy` to push to [Asset Manager Github page](https://buda-base.github.io/asset-manager)
